jQuery Blink Effect

This will cause any element to blink, with the option to provide a duration for itsÂ disappearance. 
$.fn.blink = function(time) {
	var time = typeof time == 'undefined' ? 200 : time;
	$(this).hide(0).delay(time).show(0);
}

// Example
$('.something').blink();

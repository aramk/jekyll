I'm happy to say that Crayon Syntax Highlighter is finally complete. This site is running version 1.0.0 and there doesn't appear to be any hiccups so far.

You can download the plugin from here:

http://wordpress.org/extend/plugins/crayon-syntax-highlighter/

Here's a preview in both themes in two languages, and the shortcodes that did the magic:

$[crayon url="https://raw.github.com/gist/1131372/8dc9a498e2993c1c673d95d1857e821f8f9230c8/sample.php" /] 



$[crayon theme="twilight" lang="objc"]
#import 
@interface Forwarder : Object {
    id recipient; // The object we want to forward the message to.
}
// Accessor methods.
- (id) recipient;
- (id) setRecipient:(id)_recipient;
@end
[/crayon]$
